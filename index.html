Bear Chat

Social Media Chat Platform

Bear Chat is a full stack social media chat platform that allows users to message others in his or her friend circle and share content or images that can be seen by others. It uses Go for backend, SQL as the database storing user information, AWS Neptune a graph database for storing friends' network, and dockers to provide different micro-services. Bear Chat has secure authentication procedures, email confirmation and recovery protocols. The frontend is a simple UI that's still being worked on, and it's currently deployed on a Virtual Machine hosted on EC2.

Go | SQL | AWS Neptune | Docker

Pediatric Pancreatitis Assistant

Health Management iOS App

Pediatric Pancreatitis Assistant is an iOS app designed to help patients with pediatric pancreatitis disease to track their health conditions and symptoms and provide them with a glossary of professional medical resources from reliable sources. Me and my team created this app for a UCSF client using Swift UI with firebase as backend and won third place in UCSF iHackHealth hackathon. I led a group of four designing and building the glossary section for the app, in charge of managing the database that stores several professional medical resources.

Firebase | Swift UI

Gitlet

Version Control System

Gitlet is a version control system built in Java that is modeled off Git. It utilizes file serialization, file and folder structures, and data structures and algorithm design to minimize runtime and memory usage. Gitlet supports features including adding/removing files for during staging, commits, checkouts, branching, resets, logs, and merging with an efficient cross-merging algorithm. For academic violation reasons, the code isn't public.
